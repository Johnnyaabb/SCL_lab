# SCL城市研究所资源库

## 技术选型

- 静态前端：`index.html` + `styles.css` + `data.js` + `app.js`，无需构建工具。
- 数据来源：从 `favorites_2026_6_27.html` 的“城市科学实验室”“城市研究机构”“城市研究期刊”目录筛选，并补充全球范围内城市研究相关实验室、研究中心与期刊。
- 最新资讯：浏览器端一键读取机构官网、Crossref、arXiv 与公开 RSS；跨域失败时回退到来源的“最新入口”。
- 研究资料下载：整理小红书账号提到的城市分析、街道设计、城市更新与公共空间资料，可持续补充下载链接。
- 视觉方向：采用顶部导航、分类筛选、密集卡片和简洁留白，支持中英文切换。

## 开发计划

1. 收藏夹解析：抽取目标目录，去掉不符合城市研究主题的公司、软件、课程、招聘、单篇文章等。
2. 全球检索：补充北美、欧洲、亚洲、非洲、拉美、南亚等地区的城市研究实验室、研究中心和核心期刊。
3. 来源建模：将实验室、研究机构、期刊统一写入 `SOURCES`，保留原始链接、动态入口、标签和筛选理由。
4. 首页资讯：提供“刷新最新资讯”按钮，拉取论文与动态入口卡片。
5. 资源库：按实验室、机构、期刊分类展示，支持关键词搜索和直接跳转官网。
6. 研究资料：维护 `RESEARCH_MATERIALS` 数组，展示资料封面、简介、标签和来源入口。
7. 本地验证：用静态服务器预览，检查桌面和移动端布局、交互和链接。

## 后期维护

- 新增来源：编辑 `data.js` 的 `SOURCES` 数组。
- 新增研究资料：编辑 `data.js` 的 `RESEARCH_MATERIALS` 数组，并将封面图放入 `assets/research-materials/`。
- 新增期刊：优先填写 `issn`，系统会尝试用 Crossref 获取最新论文。
- 新增机构动态：填写 `latestUrl`；如果站点有 RSS/Atom，可增加 `feed` 和 `feedKind`。
- 调整筛选：在 `EXCLUDED_SAMPLES` 中补充排除原则，在来源条目的 `includeReason` 中记录纳入原因。

## 本地预览

直接打开 `index.html` 可以查看页面。为了让浏览器网络请求表现更稳定，推荐运行：

```bash
python3 -m http.server 8080
```

然后访问 `http://127.0.0.1:8080/`。
